# 🏏 IPL Winning Team Prediction Using Machine Learning

## 📌 Project Overview

The **IPL Winning Team Prediction** project is a Machine Learning-powered web application that predicts the probability of an IPL team winning a match based on real-time match conditions. By analyzing historical IPL match data, the model estimates the winning chances of both competing teams using key match features.

The project combines **Machine Learning**, **Python**, and a **web interface** to provide users with an interactive and user-friendly prediction experience.

---

## 🚀 Features

- 🏏 Predicts the winning probability of both IPL teams.
- 📊 Uses historical IPL match data for training.
- 🤖 Machine Learning model for accurate predictions.
- 🌐 Interactive web application for easy user input.
- 📱 Simple, clean, and responsive user interface.
- ⚡ Instant prediction results with winning percentages.

## 🎯 Input Parameters

Users provide the following match details:

- Batting Team
- Bowling Team
- Match City
- Target Score
- Current Score
- Overs Completed
- Wickets Lost

The application calculates:

- Runs Left
- Balls Left
- Current Run Rate (CRR)
- Required Run Rate (RRR)

These features are passed to the trained Machine Learning model for prediction.

---

## 📈 Output

The application displays:

- Winning Probability of Batting Team
- Winning Probability of Bowling Team
- Prediction Percentage

Example:

```
Chennai Super Kings : 67%

Mumbai Indians : 33%
```
